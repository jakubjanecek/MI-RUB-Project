Class PascalParser::SymbolTable
In: lib/pascalparser/symbol_table.rb
Parent: Object

Implements a simple parser‘s symbol table for storage and retrieval of program identifiers. It also provides type information (integer or real) and memory address assignment.

Public Instance methods

find(identifier)

Returns ParserSymbol for given identifier if it is in the table or nil.

put(identifier, symbol)

Puts a new ParserSymbol for given identifier into the table if it is not present already.

type(type)

Assigns the type to all untyped symbols in the table. It also assigns an address to them.
